Telementor is the fastest and easiest way to Send Telegram Messages using Elementor Form.
 
== Description ==
 
Telementor let you connect you Elementor Form Modules with a Telegram BOT using API. By using Telementor you'll be able to Send Messages to Telegram: Groups, Channels and User using BOT API ( https://core.telegram.org/bots/api ).
Plus, Telementor can easly grab content from Elementor Form Fields (es. Email, Name etc ) and send it trought the Message by just simply using [field id=""] shortcode.

Telementor uses Telegram /SendMessage endpoint ( https://core.telegram.org/bots/api#sendmessage )

It's fully compatible with Elementor Form.

*To use this Addons you need to have an actived Elementor Pro License in order to use Elementor Form.
 
== Installation ==
 
Manual Install
 
e.g.
 
1. Upload `telementor.zip` to the `/wp-content/plugins/` directory
2. Extract 'telementor.zip' into the plugins directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Have Fun!
